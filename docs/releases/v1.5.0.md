## Highlights

- 최초 설치 후 나타나는 구독 안내(페이월)에 닫기 버튼을 추가해 자유롭게 닫고 앱을 시작할 수 있습니다.
- 광고 제거 전용(AdFree) 프로모션을 폐기하고 프로모션 혜택을 Pro 하나로 일원화했습니다.

---

## Added

- 온보딩 페이월에 닫기 버튼 노출 — 최초 설치 시 표시되는 페이월에서도 닫기 버튼이 보이며, 닫기는 "무료로 계속"과 동일하게 동작 (`paywall`)

## Removed

- 광고 제거 전용(AdFree) 프로모션 폐기 — 프로모션 혜택을 Pro로 일원화하고 광고 표시 기준을 Pro 여부로 통일 (`promotion`)
  - 프로모션 코드는 이제 Pro만 제공합니다. 기존에 AdFree 프로모션을 발급받은 사용자는 없어 영향 범위가 없으며, 서버가 AdFree 응답을 계속 내려도 클라이언트가 무시하므로 구버전 호환에 영향이 없습니다.
