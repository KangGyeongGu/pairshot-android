## Added

- 내보내기 프리셋 시스템 신설 — 합성 설정·워터마크·이미지 포맷을 한 슬롯으로 저장하고 그리드 액션바 한 번으로 적용. 무료 2슬롯·Pro 4슬롯, 길게 눌러 이름 변경/삭제. paywall 문구·체험 기간 안내도 함께 정리 (`export-settings`)
- 페어 미리보기 화면에 하단 액션바 도입 — 공유/기기 저장/AFTER 삭제/완전 삭제를 한 곳에서. 광고는 상단 배치로 이동 (`pair-preview`)
- 첫 기기 저장 직후 인앱 리뷰 요청 다이얼로그 노출. 페어 미리보기 모달 액션 수행 후 모달이 자동으로 닫힘 (`app`)
- 앨범 상세 상단 메뉴에 "페어 추가" 진입점 추가 — 페어가 이미 있는 상태에서도 이 메뉴로 PairPicker 진입 가능 (`album`)
- 앨범 상세 정렬 행 좌측에 선택 모드 진입 아이콘 버튼 추가 — 길게 누르지 않아도 빈 선택 상태로 모드 진입 (`album`)

## Changed

- 페어 카드 그리드를 공용 컴포저블로 통합 — 홈/앨범 상세/페어 추가 피커가 같은 그리드를 사용하며, 광고 슬롯·날짜 섹션 헤더·정렬이 세 화면 공통 규격으로 정렬됨. 앨범 상세/피커에도 날짜 섹션 헤더가 표시됩니다 (`ui`)
- 앨범 상세 정렬 토글을 홈과 동일한 "최신순 \| 오래된순" dual-label UI로 통일 (`album`)
- 앨범 상세에서 카메라 진입 전에 paywall 게이트를 적용 — 무료 사용자의 오늘 5개 페어 한도가 초과되면 카메라 진입 없이 paywall로 분기 (홈과 동일 정책) (`album`)
- 페어 선택 모드에서 마지막 항목 해제 시 선택 모드 자동 종료 — 홈/앨범 상세 두 화면이 같은 정책으로 통일 (`ui`)
- 앨범 상세 선택 액션바에 빈 선택 가드 추가 — 0개 선택 상태에서 공유/저장/삭제/내보내기 버튼이 흐리게 비활성 처리, 누를 수 없음 (`album`)
- 삭제·확인 모달 6종을 같은 하단 슬라이드 시트 UX로 통일 — 홈 페어 삭제, 홈 앨범 삭제, 앨범 상세 페어 삭제, 앨범 상세 앨범 자체 삭제, 페어 미리보기 페어 삭제, 페어 미리보기 AFTER 삭제 (`ui`)
- 결제 화면 "무료 버전으로 계속" 링크의 시인성 강화 — 흰색 + SemiBold 굵기로 강조. "구매 복원"과 위치 교체 (`paywall`)

## Fixed

- 갤러리 이미지 저장 진행률이 이미지 단위로 누적 표시되도록 수정 — 이전 0/12 → 0/36 흐름이 0/N 단조 증가로 안정 (`export`)

## Internal

- 공유/저장 흐름을 단일 파이프라인(`ExportRepository`)으로 통합하고 작업 계획(`ExportPlan`)+ZIP 압축 단계를 별도 진행률 단계로 분리 (`export`)
- `:core:ui`에 공용 `SelectionState` 도입 — 세 ViewModel(홈/앨범 상세/페어 추가)의 선택 상태 표현을 통일 (`ui`)
- 앨범 페어 연결 동작을 use case로 캡슐화 — `RemovePairsFromAlbumUseCase`/`AddPairsToAlbumUseCase` 신설, ViewModel이 repository 직접 호출하지 않도록 정리 (`album`)
- 정렬 토글 컴포저블(`SortOrderDualLabel`)을 `:core:ui`로 공용화하고 기존 `SortOrderToggleButton` 제거 (`ui`)
- ArchUnit `L-11`(AdsUi 모듈 경계) 위반 수정 — `PairCardGridSection`을 `:core:ui`로 이동, 광고 컨텍스트는 `:core:ads-ui`의 `PairCardNativeAdSlot` 헬퍼로 분리 (`arch`)
- ktlint 0.50.0 + ktlint-gradle 11.6.1 도입으로 detekt-formatting과 룰 정렬 — `.editorconfig`가 단일 진실 (`quality`)
- 앨범 선택 모드 진입 버튼 영어 번역 추가, R8 release 빌드의 `NoNullnessRewrite` missing class 경고 무시, Play core/review 의존성 baseline 갱신, `WatermarkConfig` serializer ABI dump, OWASP CVE-2020-29582·logback false positive suppress (`i18n` / `build` / `deps` / `api` / `security`)
