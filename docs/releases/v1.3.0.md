## Highlights

- 합성 시 BEFORE/AFTER 레이블을 사진 외부 테두리에 새기는 방식 추가 — 이미지 영역을 가리지 않고 레이블이 위치한 변만 자동으로 두꺼워짐, BEFORE/AFTER 각각 위·아래 좌/중/우 6 칸 자유 배치(비대칭 허용).
- AFTER 촬영 중 BEFORE 원본을 길게 누르면 전체화면 미리보기.
- PRO 구독자에게도 "구독 옵션 보기" 진입점 노출 + paywall 자동 닫힘 방지.

## Added

- 합성 레이블 사용 방식 토글 — "이미지 내부" / "테두리 내부" 분기. 테두리 내부 모드는 글자가 위치한 변(VERTICAL 합성의 경우 가운데 갭 포함) 만 자동 확장하고 나머지 변은 사용자 설정 두께 유지. BEFORE/AFTER 각각 2×3 시각적 picker, 모드 전환 시 테두리 자동 ON · 잠금(레이블 ON 일 때만) (`combine`)
- AFTER 촬영 중 BEFORE 원본 길게 누르기 전체화면 미리보기 + 카드 강조 색 · 튜토리얼 단계 · 설정 튜토리얼 흰 테두리 정리 (`camera`)
- PRO 구독자도 구독 옵션 보기 진입점 노출, paywall 자동 닫힘 방지 (`settings`)

## Internal

- Play Console internal 트랙 자동 업로드 GitHub Actions 워크플로우 — tag v*.*.* push 시 :app:bundleRelease → r0adkll/upload-google-play 액션으로 draft 상태 업로드. `local.properties` 동적 생성 방식이라 모듈 build.gradle.kts 무변경 (`ci`)
- 합성 설정 토글 row 4 곳을 `SegmentedToggleRow` 공통 컴포저블 1 개로 통합, `PositionPicker3x3` → `PositionPickerGridRow` 로 일반화하여 6/9 칸 picker 통합, 도메인 보정 로직 (`CombineConfig.withLabelPlacement` · `LabelAnchor.coercedForBorderPlacement`) 을 `:core:model` 로 이동, fontSize 계산을 referenceImageDim 기준으로 통일 (`combine`)
- README 저작권 표기를 사업자명 (NomadLabs) 으로 갱신 (`readme`)
