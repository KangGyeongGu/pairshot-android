## Highlights

- 앱 자체 텍스트 크기 설정 신설(작게 / 보통 / 크게 / 아주 크게). 기기의 글꼴 크기·디스플레이 크기 슬라이더는 더 이상 PairShot UI 에 영향을 주지 않고, 사용자가 선택한 인앱 값으로 일관 적용됩니다.
- 튜토리얼 시스템 강화 — 풍선 모달을 표준 컴포넌트 기반으로 재구성, 현재 단계를 디스크에 저장해 강제 종료 후에도 멈춘 지점부터 이어보기, BEFORE 크게보기 닫기 단계 신설로 다음 안내가 미리보기를 가리지 않게 함.
- 카메라 설정 시트 가독성 개선 + 오버레이 사용 OFF 시 슬라이더가 깔끔하게 접힘.

## Added

- 앱 자체 텍스트 크기 설정 — 4 단계(SMALL / NORMAL / LARGE / EXTRA_LARGE). DataStore 에 저장되고 `PairShotTheme` 가 진입점에서 `LocalDensity` 를 인앱 값으로 고정해 OS 글꼴/디스플레이 크기 슬라이더 영향 0. CameraX `PreviewView` · AdMob 배너 등 native View 는 시스템 density 그대로 (의도된 동작) (`ui`)
- 튜토리얼 BEFORE 크게보기 닫기 안내 단계 — long press 로 켠 후 빈 영역 탭(또는 백버튼)으로 닫도록 학습. 이후 AFTER 촬영 안내 모달이 미리보기와 겹치지 않게 됨 (`tutorial`)
- 튜토리얼 단계 영속성 — 현재 단계를 DataStore 에 저장하고 앱 시작 시 자동 복원. process death · 강제 종료에서 멈춘 위치 그대로 재개 (`tutorial`)

## Changed

- 표준 레이아웃 기반 — ActionBar / TopBar / BottomSheet / TopMenu 등의 `.height(fixed)` 를 `.heightIn(min=)` 으로 전환해 최소 터치타깃은 보장하면서 인앱 텍스트 크기가 커져도 한국어 긴 라벨이 잘리지 않고 자연스럽게 wrap (`ui`)
- 설정·내보내기 화면의 모든 리스트 행을 M3 `ListItem` 기반의 `PairShotListItem` / `PairShotSwitchListItem` 로 통합 (`ui`)
- 튜토리얼 풍선을 표준 `androidx.compose.ui.window.Popup` + 자체 `PopupPositionProvider` 로 재구성. 앵커 좌우 중앙 정렬 + 위·아래 공간 넓은 쪽 자동 배치 + status / navigation bar inset clamp (`tutorial`)
- 튜토리얼 회피 영역을 `actionAnchor` 우선으로 변경해 CAMERA_PORTRAIT 등에서 셔터·스트립 가림 차단 (`tutorial`)
- 튜토리얼 카메라 회전 안내 문구 — 키워드 (세로로 / 좌측으로 / 우측으로) 굵게 + primary color 강조 (`tutorial`)
- 선택 모드 안내 문구 톤 조정 — 명령형 → 가능 표현 (`tutorial`)

## Fixed

- 카메라 설정 시트 가독성 — `PairShotGlassTokens.surfaceColor` alpha `0x99` → `0xE6`. 프리뷰가 비쳐서 설정 아이콘이 흐릿하던 회귀 해소 (`camera`)
- 오버레이 슬라이더 접기 — `OverlayAlphaSlider` 가 overlayEnabled 토글에 맞춰 `AnimatedVisibility` 로 완전히 접힘. 이전엔 disabled 처리만 해 트랙 라인이 흐릿하게 남아 있었음 (`camera`)
- 튜토리얼 마지막 안내 모달이 설정 항목과 겹쳐 보이던 spotlight 뚫림 회귀 차단 (`tutorial`)
- Popup / Dialog / DropdownMenu / ModalBottomSheet 안에서 인앱 텍스트 크기가 안 먹던 회귀 — 모달은 새 `ComposeView` 를 만들어 `LocalDensity` 가 시스템 값으로 리셋된다. `LocalAppTextScale` + `ProvideAppTextScaleDensity` helper 도입 후 wrapper 5종 모두에 적용해 일관 차단 (`designsystem`)
- 파일명 prefix 다이얼로그 에러 영역 클리핑 방지 (`settings`)
